# 冒险岛阿尔泰服找王辅助工具 (MapleStory Artale Boss-Hunting Helper)

这是一个专门为《冒险岛阿尔泰亚服（Artale）》设计的找王（Boss 刷新）辅助工具。它通过高精度的内置本地 OCR 检测屏幕右下角战斗日志的经验值增加文字，并进行去重智能计数。如果触发 Boss 刷新（屏幕中央出现“能感受到邪惡的氣息。”），悬浮窗会变成红色并闪烁提醒。

## 功能特性
1. **自动跟随窗口**：启动后自动获取游戏客户端位置，悬浮窗随游戏窗口拖动同步移动。
2. **完全不影响操作**：默认处于鼠标穿透状态（无法被鼠标点击到），绝对不卡视角或阻碍点击。
3. **三种极简状态**：
   - 击杀数不满 10 只：黄色高对比度边框文字 `KILLS: X / 10`。
   - 击杀数满 10 只：绿色醒目文字 `KILLS: 10 / 10`。
   - 出现 Boss 警报：红色闪烁并显示 `⚠️ BOSS SPAWNED! ⚠️`。

## 快捷键说明 (热键全局生效)

| 快捷键 | 功能 | 说明 |
|:---:|:---|:---|
| **`F8`** | **一键重置计数器** | 换线后按此键，计数归零并清除 Boss 警告状态。 |
| **`F9`** | **切换解锁 / 锁定位置** | 解锁后悬浮窗会显示背景色，此时可以用鼠标左键**按住拖动**到任意位置（例如游戏窗口的顶部或空闲区）。再次按 F9 锁定悬浮窗并重新启用鼠标穿透。 |
| **`F10`** | **暂停 / 恢复检测** | 临时挂机或不打怪时可以按此键暂停，避免误统计。 |

## 安装与运行

### 1. 推荐将当前项目目录设置为 active workspace
在 AI 助手交互界面或者您的 IDE 中，建议将本项目的目录设置为活跃工作区：
`C:\Users\suhao\.gemini\antigravity\scratch\artale_boss_helper`

### 2. 安装依赖包
在您的命令行中，使用 `pip` 安装本工具所需的依赖包（已经为您提前安装完成，如需在其他环境部署可运行此命令）：
```bash
pip install pyautogui pillow numpy opencv-python keyboard winocr
```

### 3. 运行程序
确保游戏窗口处于打开状态，在控制台运行以下命令：
```bash
python artale_boss_helper.py
```

## 模拟测试 (Mock Mode)

如果您在没有游戏运行的机器上测试，可以启动本工具的**仿真模拟模式**：
```bash
python artale_boss_helper.py --mock
```
- **模拟模式行为**：
  - 会模拟产生一个游戏窗口。
  - 前面几次扫描中，会模拟加载第一张图（有击杀经验），您会看到计数器逐渐累加。
  - 约第 8 秒后，模拟加载第二张图（有 Boss 警报文字），您会看到悬浮窗红字闪烁提示 `⚠️ BOSS SPAWNED! ⚠️`。
  - 期间您可以按下 `F9` 拖拽悬浮窗，或者按下 `F8` 将其重新归零测试。
